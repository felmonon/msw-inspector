# Support

## Where to ask

- Use [GitHub issues](https://github.com/felmonon/msw-inspector/issues) for reproducible bugs, incorrect matches, missed API calls, stale mock false positives, and feature requests.
- Use GitHub Discussions if they are enabled for broader usage questions or workflow advice.
- Use the [security policy](./SECURITY.md) for vulnerability reports.

## Before opening an issue

Please include enough detail to reproduce the behavior:

- `msw-inspector-cli` version
- Node.js and npm versions
- the command you ran
- relevant `--handlers`, `--sources`, `--exclude`, and `--base-url` values
- a small handler snippet and API-call snippet when matching looks wrong
- text output or a trimmed JSON report from `--format json`

## Maintainer expectations

This is an open-source project maintained as availability allows. There is no guaranteed response time, but clear reproductions and small examples make issues much easier to investigate.
